# BJ's QT Code

> Paddle Inference Qt SQlite3 部署检测回溯系统

--- 
为解决工厂产品检测与回溯问题，创建此项目。主要基于Paddle Inference在C++环境中的部署实现识别检测，通过SQlite在C++环境中的操作实现数据库管理利用Qt制作GUI并联系各个部分。



目录：

* 使用Paddle Inference部署视觉识别模型
* Qt GUI页面部署与响应
* 内嵌Sqlite3实现数据库管理
